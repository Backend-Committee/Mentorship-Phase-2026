# Mentorship-Phase-2026

## Library Management System

This is a Python-based Library Management System that allows users to manage books and users in a library. The system supports adding users, adding books, borrowing books, returning books, and viewing available books.

---

## Features

- Add new users to the library.
- Add new books to the library.
- Borrow books for registered users.
- Return borrowed books.
- View all available books.
- Display all registered users and their borrowed books.
