# Mini Blog - Django Application

A simple blog application built with Django, featuring a clean interface for viewing and managing blog posts.

## Project Description

Mini Blog is a lightweight Django web application that allows users to:
- View a list of blog posts on the home page
- Read individual blog posts
- Browse blog content with a responsive design

## Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

## Setup Instructions

1. **Navigate to the project directory:**
   ```bash
   cd task_4
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Navigate to the Django project:**
   ```bash
   cd mini_blog
   ```


## Running the Application

1. **Start the development server:**
   ```bash
   python manage.py runserver
   ```

2. **Access the application:**
   - Open your web browser and navigate to `http://127.0.0.1:8000/`
