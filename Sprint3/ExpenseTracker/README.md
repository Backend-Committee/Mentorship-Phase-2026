# Expense Star

**Expense Star** is a terminal-based expense tracker built with **Python**, designed to help you manage your finances directly from the command line in a fast and distraction-free way.

It supports tracking expenses and income, managing multiple accounts, and transferring money between them through an interactive CLI.

---

## Features

* Track expenses and income
* Manage multiple bank accounts
* Transfer money between accounts
* Built with Object-Oriented Programming (OOP)
* Persistent storage with SQLite
* Interactive CLI powered by InquirerPy
* Styled terminal output using ASCII art and colors

---

## CLI Experience

Expense Star runs entirely in the terminal and provides a dynamic, menu-driven interface:

* Navigate options using arrow keys
* Input validation and guided prompts
* Clear and structured visual feedback
* No GUI or browser required

---

## Tech Stack

* Python 3
* SQLite
* InquirerPy
* pyfiglet
* termcolor
* prompt_toolkit

---

## Installation

1. Clone the repository

```bash
# git clone <this repo>
cd expense-star
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

---

## Running the App

```bash
python main.py
```

You will be greeted with the Expense Star banner and an interactive menu.
